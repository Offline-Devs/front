# Noshirvani Academy Frontend

scaffold معماری فرانت مطابق backend موجود. همه routeها، typeها، API moduleها، store/cache، schemaها و componentهای لازم ساخته شده‌اند و داخل هر فایل مسئولیت آن نوشته شده است.

```bash
Copy-Item .env.example .env.local
npm install
npm run dev
```
